# 🔎 SQL Retrieval

## 📌 Descripción

El sistema implementa recuperación de información mediante consultas SQL ejecutadas dinámicamente desde el workflow de n8n.

---

# ⚙️ Objetivo

Permitir que el agente IA:
- consulte información estructurada
- recupere registros empresariales
- responda preguntas basadas en datos reales

---

# 🧠 Integración IA

El modelo conversacional interpreta la intención del usuario y utiliza herramientas SQL para obtener información relevante.

---

# 🔄 Flujo de Recuperación

```text
Pregunta Usuario
    ↓
Agente IA
    ↓
Herramienta SQL
    ↓
Consulta MySQL
    ↓
Resultado
    ↓
Respuesta Conversacional
```

---

# 🗃️ Tecnologías Utilizadas

- n8n
- MySQL
- Railway
- Cohere
- Telegram

---

# 🚀 Beneficios

- consultas dinámicas
- integración empresarial
- respuestas basadas en datos
- automatización inteligente
